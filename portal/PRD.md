# Codtech Internship Platform - PRD

## Project Overview
A professional internship portal built with Next.js 14, providing domain-specific project tracking for students.

## Key Features
1. **User Authentication**: Secure login and registration for interns.
2. **Dynamic Dashboard**: Personalized view of 30+ unique projects based on the user's assigned course/domain.
3. **Professional UI**: High-end blue theme with glassmorphism effects and responsive layout.

## Tech Stack
- **Frontend**: Next.js (App Router), Tailwind CSS, Framer Motion.
- **Backend**: Next.js API Routes, MySQL (via mysql2).
- **Deployment**: Hostinger (Git Deployment).

## User Flows
1. **Sign Up**: User registers with name, email, and selected course.
2. **Login**: User signs in to access their dashboard.
3. **Dashboard**: User views projects specific to their domain (e.g., Python, React, Java).
